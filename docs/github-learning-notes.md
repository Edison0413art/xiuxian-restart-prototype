# GitHub 参考学习笔记

参考仓库：VickScarlet/lifeRestart。

## 可复用结构

- `Life` 作为总控，串联属性、天赋、事件、成就和角色。
- `Talent` 负责随机抽取、稀有度、互斥、替换和触发条件。
- `Event` 负责事件池、条件筛选、分支事件和效果结算。
- `Property` 负责属性读写、最高/最低统计、结算评分和死亡判定。
- 年龄表提供每一年候选事件，事件表再通过条件和权重决定实际触发内容。
- 结算页读取本局最高属性和选中天赋，形成总结。

## 本原型的改造

- 保留“属性 + 天赋 + 事件池 + 条件过滤 + 总结”的骨架。
- 把自动逐年播放改成每个时间节点随机 2-4 个选择。
- 把原始五维属性改为：灵根、悟性、体魄、家底、气运、颜值。
- 增加动态数值：境界、修为、灵石、名声、左韩注视度。
- 不使用原项目的 LayaAir UI，改成轻量 HTML/CSS/JS 手机网页。
- 主流程本地运行，DeepSeek 只作为“请天道加戏”的可选旁白。

## 热梗素材处理

- 热词素材分为网络年度词、短视频口头禅、小说套路、跨文体串台四层。
- 游戏内不追求单个梗复读，而是把梗作为事件标签和文本变量混搭。
- 首版事件池为 1000 条模板生成样例，结局池为 100 个。
